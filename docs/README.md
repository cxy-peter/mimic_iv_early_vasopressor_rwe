# Documentation - MIMIC-IV Early Vasopressor RWE

This folder keeps four human-facing report versions for different audiences.

| File | Audience | Language | Purpose |
|---|---|---|---|
| `academic_report_en.md` | Academic / technical reviewer | English | Manuscript-style project report with research question, methods, results, interpretation, limitations, and reproducibility notes. |
| `academic_report_zh.md` | Academic / technical reviewer | Chinese | 中文学术版报告，适合给老师、同学、技术面试官或 RWE/生统同事看。 |
| `presentation_brief_en.md` | Medical team / hiring manager / business stakeholder | English | Short briefing version focused on why the project matters, group definitions, workflow, key results, and resume wording. |
| `presentation_brief_zh.md` | 医学团队 / 老板 / 在职人士 | Chinese | 中文汇报版，强调研究问题、比较组、关键结果、管理层解释和简历表达。 |

The Markdown format is used so the files render cleanly on GitHub and can be easily converted to DOCX/PDF later if needed.
